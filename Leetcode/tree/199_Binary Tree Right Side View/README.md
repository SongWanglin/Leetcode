**199. Binary Tree Right Side View**

Given a binary tree, imagine yourself standing on the _right_ side of it, return the values of the nodes you can see ordered from top to bottom.

**Example:**

**Input:** [1,2,3,null,5,null,4]

**Output:** [1, 3, 4]

**Explanation:**

   1            &lt;---

 /   \

2     3         &lt;---

 \     \

  5     4       &lt;---
