**171. Excel Sheet Column Number**

Given a column title as appear in an Excel sheet, return its corresponding column number.

For example:

    A -&gt; 1

    B -&gt; 2

    C -&gt; 3

    ...

    Z -&gt; 26

    AA -&gt; 27

    AB -&gt; 28 

    ...

**Example 1:**

**Input:** "A"

**Output:** 1

**Example 2:**

**Input: **"AB"

**Output:** 28

**Example 3:**

**Input: **"ZY"

**Output:** 701
